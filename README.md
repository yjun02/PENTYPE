# 🧩 PENTYPE (펜타입)

> **"기존 MBTI를 넘어선 5번째 지표, 당신의 경계 유형을 찾아보세요."**

PENTYPE은 기존의 4가지 MBTI 지표(E/I, S/N, T/F, J/P)에 '나와 타인 사이의 벽'을 정의하는 **W/O(Wall/Organic)** 지표를 추가하여, 총 32가지의 더욱 정밀한 성격 유형을 분석해주는 심리 테스트 서비스입니다.

## ✨ Key Features

- **5번째 지표 W/O**: 개인의 프라이버시와 경계를 대하는 태도를 측정하는 신규 지표 도입.
- **듀얼 트랙 시스템**:
    - **Full Track**: 50문항을 통한 정밀 측정 (MBTI + W/O)
    - **Fast Track**: 기존 MBTI를 알고 있다면 W/O 지표만 빠르게 확인 (10문항)
- **정밀 타이브레이커**: 점수가 50:50인 경우, 답변의 강도(4, 5번 선택지)를 분석하여 확실한 성향을 도출.
- **비주얼 리포트**: 2단 분할 레이아웃(PC)과 모바일 최적화 레이아웃을 통해 한눈에 들어오는 분석 결과 제공.
- **결과 이미지 저장 및 공유**: `html2canvas`를 활용한 고화질 결과 카드 저장 기능 및 암호화된 URL 링크 공유 지원.

## 🛠 Tech Stack

- **Frontend**: React (v19)
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Design System)
- **Libraries**: `react-router-dom`, `html2canvas`

## 🚀 Getting Started

```bash
# 의존성 설치
npm install

# 로컬 개발 서버 실행
npm run dev
```
